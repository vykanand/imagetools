API Documentation.
Integrate ImageTools.

RESTful API for background removal, image resizing, filters, and more.

Base URL: http://your-host:5001

AUTH POST /api/auth/signup

Create a new user account.

Body (JSON):
{ "email": "user@example.com", "password": "yourpassword", "name": "Optional Name" }
Example:
curl -X POST http://localhost:5001/api/auth/signup -H "Content-Type: application/json" -d '{"email":"user@example.com","password":"pass123"}'
AUTH POST /api/auth/login

Log in and receive a JWT token.

Body (JSON):
{ "email": "user@example.com", "password": "yourpassword" }
Response:
{ "token": "jwt...", "user": { "email": "...", "name": "..." } }
Example:
curl -X POST http://localhost:5001/api/auth/login -H "Content-Type: application/json" -d '{"email":"user@example.com","password":"pass123"}'
FREE POST /api/remove-bg

Remove background from an image using AI (silueta model). Returns a transparent PNG.

Parameters (multipart/form-data):
image: (file) Image to process — JPG, PNG, WebP (max 20MB)
Response:
{ "id": "uuid", "url": "/api/output/uuid.png" }
Example:
curl -X POST http://localhost:5001/api/remove-bg -F "image=@photo.jpg"
FREE POST /api/background-blur

Blur the background of an image while keeping the subject sharp. Uses AI subject detection.

Parameters (multipart/form-data):
image: (file) Image to process
blurRadius: (int) 1–30, default 10
Response:
{ "id": "uuid", "url": "/api/output/uuid.png" }
Example:
curl -X POST http://localhost:5001/api/background-blur -F "image=@photo.jpg" -F "blurRadius=15"
FREE POST /api/selfie-filter

Apply filter presets to images. Available presets: glow, warm, vintage, bw-classic, soft-glam, vivid, cool, dramatic.

Parameters (multipart/form-data):
image: (file) Image to process
preset: (string) One of: glow, warm, vintage, bw-classic, soft-glam, vivid, cool, dramatic
Response:
{ "id": "uuid", "url": "/api/output/uuid.png" }
Example:
curl -X POST http://localhost:5001/api/selfie-filter -F "image=@selfie.jpg" -F "preset=vivid"
AUTH POST /api/compose

Compose a foreground (transparent PNG) over a background with text overlay. Requires JWT authentication.

Parameters (JSON or multipart):
foregroundId: (string) ID from /api/remove-bg output
bgType: "solid" | "gradient" | "url"
bgColor: hex color (for solid)
gradientColors: JSON array of hex colors
text: overlay text (optional)
textColor, textSize, textRotation, textX, textY
Example:
curl -X POST http://localhost:5001/api/compose -H "Authorization: Bearer TOKEN" -H "Content-Type: application/json" -d '{"foregroundId":"uuid","bgType":"gradient","gradientColors":["#ff7e5f","#feb47b"],"text":"Hello"}'
FREE POST /api/store-image

Upload an image and get a shareable URL. Files are stored server-side for 24 hours.

Parameters (multipart/form-data):
image: (file) Any image format (max 20MB)
Response:
{ "id": "uuid", "url": "/api/output/uuid.png" }
Example:
curl -X POST http://localhost:5001/api/store-image -F "image=@photo.jpg"
FREE GET /api/output/:filename

Retrieve a processed image by filename. Used to access results from any API endpoint.

Parameter:
filename: (string) e.g. "uuid.png"
Example:
curl http://localhost:5001/api/output/uuid.png --output result.png
FREE GET /api/auth/me

Get current authenticated user info. Requires JWT token in Authorization header.

Example:
curl http://localhost:5001/api/auth/me -H "Authorization: Bearer TOKEN"
All API endpoints return JSON. Authentication required endpoints return 401 if token is missing or invalid.
